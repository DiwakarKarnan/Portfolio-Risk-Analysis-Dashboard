# Portfolio Risk Analysis Dashboard

An interactive **Power BI** dashboard that simplifies portfolio performance monitoring and financial risk analysis using **near real-time market data** from Yahoo Finance.

The project demonstrates an end-to-end analytics workflow, integrating **Python**, **Google Colab**, **Google Drive**, and **Power BI** to automate data collection, processing, and visualization.

---

# Project Objective

Many financial platforms provide a large amount of market information, which can be overwhelming for long-term investors.

This project was designed to provide a **Purpose-Built and simplified analytics dashboard**, allowing investors to focus on the financial metrics that are most relevant to their investment strategy while reducing unnecessary information.

Rather than replacing existing investment platforms, this dashboard demonstrates how business intelligence can present financial insights in a more focused and user-friendly way.

# Financial Metrics

The dashboard calculates and visualizes several important investment metrics including:

- Annual Return
- Annual Volatility
- Daily Returns
- Sharpe Ratio
- Portfolio Beta
- Value at Risk (VaR 95%)
- Rolling Volatility
- Correlation Matrix
- GARCH (1,1) Volatility Model

---

# Data Pipeline

```

Yahoo Finance
↓
Python (Google Colab)
↓
Financial Data Processing
↓
Store Processed Data in Google Drive
↓
Power BI Data Refresh
↓
Interactive Dashboard

```

The dashboard uses **near real-time** market data.

Running the Python notebook retrieves the latest financial data from Yahoo Finance, processes portfolio metrics, stores the processed data in Google Drive, and allows Power BI to display updated insights after a refresh.
